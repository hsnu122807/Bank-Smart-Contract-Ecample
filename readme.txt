操作流程:
1.將所有檔案下載至同一個資料夾內
2.維持網路的連接
3.打開Ganache1並確保port為local host的8545
4.打開index.html
5.按下部屬新合約按鈕
6.選擇登入帳戶
7.存款1並按下執行
8.購買定存在金額欄位填1，期數欄位填12並按下購買
此操作會扣除銀行ETH餘額並將ETH轉入定存，在帳戶狀態將顯示如下:
以太帳戶餘額 (wei): x
銀行ETH餘額 (wei): 0
銀行定存 (wei): 1000000000000000000
銀行定存期數: 12
銀行定存利息: 0
9.按下定存期滿按鈕
此操作會將定存本金轉回銀行ETH帳戶當中，並給予利息120000000000000000(wei)，帳戶狀態顯示如下:
以太帳戶餘額 (wei): x
銀行ETH餘額 (wei): 1000000000000000000
銀行定存 (wei): 0
銀行定存期數: 0
銀行定存利息: 120000000000000000
10.購買定存在金額欄位填1，期數欄位填12並按下購買
11.在已完成期數欄位填6並按下提前解約按鈕
此操作會將定存本金轉回銀行ETH帳戶當中，並給予利息60000000000000000(wei)，帳戶狀態顯示如下:
以太帳戶餘額 (wei): x
銀行ETH餘額 (wei): 1000000000000000000
銀行定存 (wei): 0
銀行定存期數: 0
銀行定存利息: 180000000000000000


說明:
帳戶狀態會顯示ETH帳戶餘額以及智能合約中銀行的ETH餘額
由於此智能合約並不會真正的發放ETH作為利息
因此本作業將定存金額及利息與原本的銀行存款分開
避免銀行ETH餘額的混淆
在操作上設有一些限制
定存的購買是使用銀行ETH的餘額進行購買
因此若銀行ETH的餘額不足則無法購買
購買後會將ETH轉入定存帳戶並從銀行ETH扣除
!!!注意:當已經持有定存的時候無法再購買新的一筆定存!!!
而不論是定存期滿或提前解約都會將定存的本金從定存帳戶轉回銀行ETH帳戶
產生的利息會額外紀錄在銀行定存利息的部分
另外要注意的是
提前解約的期數必須小於原本購買定存時所輸入的期數，否則無法解約
